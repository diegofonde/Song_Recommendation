# Song_Reommendation

## About the project
A song recommendation system partially insipired by my original Movie Recommendation project, and my Song Recommendation project done for my Machine Learning Class. This project involves a hybrid reommendation engine utilizing stacked autoencoders, RNN and a XGBoost model which is coded in Python. Along with deploying these Machine Learning algorithms, a backend and frontend will be desgined around this involving languages such as SQL, HTML, CSS and JavaScript in order to create a functional music recoomendation app.
